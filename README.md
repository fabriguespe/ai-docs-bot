# XMTP AI Docs Bot

Learn about XMTP by chatting with the [Docs](https://xmtp.org/docs/introduction) through a GPT powered bot.

## Getting started

To install dependencies:

```bash
yarn
```

To run:

```bash
yarn build
yarn start
```

To run with hot-reload:

```bash
yarn build:watch
yarn start:watch
```

### Environment

```bash
cp .env.example .env
```

then populate the environment variables accordingly

```bash
KEY=
XMTP_ENV=
KAPA_API_ENDPOINT=
KAPA_API_TOKEN=
```
